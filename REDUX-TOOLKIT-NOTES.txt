PART 1 REDUX DEMO
    npm init

    INSTALL REDUX
        npm i redux
    
    RUN
        node index / npm start          (edit script in package json)

    INSTALL IMMER                       (simplify the nested object in state in redux. simplify handling immutable data structures) 
        npm i immer

    INSTALL REDUX LOGGER                (a logger. it does what store.getState() do )
        https://www.npmjs.com/package/redux-logger
        npm i redux-logger

    INSTALL REDUX THUNK                 (a middleware use for logging , crash reporting , perform async task )
        npm i redux-thunk

    INSTALL AXIOS                       ( use for api data fetching)
        npm i axios
